🎯 OBJECTIF DU PROJET
Vision Globale
RPG Engine est un générateur de jeux de rôle procédural utilisant l'IA pour créer des univers persistants, narratifs et tactiques. Le joueur évolue dans un monde sandbox où chaque action a des conséquences permanentes.
Caractéristiques Principales

Narration IA : Scénarios générés et adaptés dynamiquement (Gemini 3 Flash Preview + futur LLM local)
Monde Persistant : 10 000+ PNJ avec mémoire, intentions et évolution autonome
Combat Tactique Deckbuilding : Inspiré de Magic: The Gathering + Into the Breach + Slay the Spire
Génération Procédurale : Monde, quêtes, PNJ, événements créés à l'initialisation
Multimodalité : Texte + images générées (pixel art 16-bit style Chrono Trigger/FFVI)
Multijoueur Futur : Monde partagé MMO-lite avec PvE coopératif et PvP arène

source venv/bin/activate
Solution permanente (à implémenter) :
Ajouter au ~/.bashrc :
bash# Auto-activate RPG Engine venv
if [ -d "$HOME/Documents/rpg-engine/venv" ]; then
    source "$HOME/Documents/rpg-engine/venv/bin/activate"
fi

### Gestion Mémoire Contextuelle (Design Cible)
```
┌──────────────────────────────────────────────────┐
│          CONTEXTE ACTIF (RAM)                    │
│  - 10 dernières actions                          │
│  - Scène actuelle complète                       │
│  - PNJ zone (5 max détaillés)                    │
│  - Quêtes actives (3 max)                        │
│  Total : ~2000 tokens Gemini                     │
└──────────────┬───────────────────────────────────┘
               │
               │ Compression toutes les 10 actions
               ▼
┌──────────────────────────────────────────────────┐
│     ARCHIVES COMPRESSÉES (Serveur)               │
│  data/saves/[session_id]/history_compressed/     │
│  - turn_0010.json : Résumé actions 1-10          │
│  - turn_0020.json : Résumé actions 11-20         │
│  Format : {                                      │
│    "summary": "Le héros a vaincu 3 gobelins...", │
│    "key_events": ["gobelin_chef_mort"],          │
│    "npcs_met": ["Elara_la_mage"],                │
│    "items_gained": ["épée_en_fer"]               │
│  }                                               │
└──────────────┬───────────────────────────────────┘
               │
               │ Reconstruction contextuelle
               ▼
┌──────────────────────────────────────────────────┐
│      PROMPT GEMINI DYNAMIQUE                     │
│  1. Résumé monde (500 tokens)                    │
│  2. Historique compressé (300 tokens)            │
│  3. Scène actuelle (800 tokens)                  │
│  4. PNJ présents (400 tokens)                    │
│  Total : ~2000 tokens → Contexte optimal         │
└──────────────────────────────────────────────────┘
```
